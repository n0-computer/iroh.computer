# blobs export

Export a blob from the internal blob store to the local filesystem {{ className: 'lead' }}


### Arguments

| name | necessity | description |
| ---- | --------- | ----------- |
| hash  | required | The hash to export |
| out  | required | Directory or file in which to save the file(s) |
| recursive  |  | Set to true if the hash refers to a collection and you want to export all children of the collection |
| stable  |  | If set, the data will be moved to the output directory, and iroh will assume that it will not change |
