# blobs download

Download data to the running node&#39;s database and provide it {{ className: 'lead' }}


### Arguments

| name | necessity | description |
| ---- | --------- | ----------- |
| ticket or hash  | required | Ticket or Hash to use |
| address  |  | Additional socket address to use to contact the node. Can be used multiple times |
| relay-url  |  | Override the relay URL to use to contact the node |
| recursive  |  | Override to treat the blob as a raw blob or a hash sequence |
| override-addresses  |  | If set, the ticket&#39;s direct addresses will not be used |
| node  |  | NodeId of the provider |
| out  |  | Directory or file in which to save the file(s) |
| stable  |  | If set, the data will be moved to the output directory, and iroh will assume that it will not change |
| tag  |  | Tag to tag the data with |
| queued  |  | If set, will queue the download in the download queue |
